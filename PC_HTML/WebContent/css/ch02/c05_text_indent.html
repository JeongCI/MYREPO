제약 사항

개발과 관련된 정책이나 업무규칙, 특정 소프트웨어나 프로그램의 사용, 데이터 사용과 관 련된 제약 등에 대한 기술 제약사항(Technical constraints), 개발할 때 적용할 업무 영 역의 표준이나 법규의 표준 적합 제약사항(Standard compliance constraints)으로 나눌 수 있다
