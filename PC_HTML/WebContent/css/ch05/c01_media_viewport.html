미디어 쿼리(media)

미디어 쿼리는 화면 크기에 따라 각각의 속성 값을 지정하여, 여러가지 화면을 구성하는 기술

@media only all and(조건문) {실행문}

  • @media: 미디어 쿼리가 시작
  • only: 미디어 쿼리 구문을 해석하세요
  • all: 미디어 쿼리 해석 해야할 대상
    • all: 모든 미디어 유형에서 사용할 CSS를 정의
    • print: 인쇄장치 에서 사용할 CSS를 정의
    • screen: 컴퓨터 스크린에서 사용할 CSS를 정의
    • tv: TV에서 사용할 CSS를 정의
    • projection: 프로젝터에서 사용할 CSS를 정의
