나의 아름다운 정원

사람들이 선호하는 꽃들은 어떤 것들일까..
아마 스스로에게 자신감을 북돋워 주는 표정있는 꽃들일 것이다.

  • 튜울립

    고백할 때 많이 쓰이는 꽃인 이유가 여기에 있어요. 영원한 사랑의 고백을 뜻하니 받는 분도, 선물하는 분도 정말 행복하겠죠? 빨간색 튤립과 비슷한 핑크색 튤립은 꽃말도 역시 비슷합니다. 사랑의 시작, 혹은 애정과 배려를 뜻해요

  • 해바라기

    중국 문화와 상징주의 측면에서 해바라기는 장수와 행운을 상징하며 길조로 여겨집니다. 중국인에게 노란색은 활력과 지성을 상징합니다.

  • 장미

    붉은 장미 - 사랑, 아름다움, 낭만적인 사랑, 용기, 존경, 열망, 열정. 붉은 장미 한 송이 - 당신을 사랑합니다
